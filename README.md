
```bash
# Clonar Repositório
$ git clone https://github.com/carlos334lima/github.git

# Seguir para o repositório
$ cd github

# Escolher Projeto que deseja rodar.

# ReactJS            # React-Native
$ cd web             $ cd mobile

$ yarn install

# Rodar projeto

# ReactJS            # React-Native
$ yarn start         $ yarn run android/ios 


#Observação:
```
